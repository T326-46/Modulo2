# Aula 05 - Renomeação de Colunas no Pandas

## Objetivo

Essa aula tem como objetivo mostrar os passos para renomear o nome das colunas em um objeto Pandas e as formas possíveis. Ao final desta aula o participante será capaz de renomear colunas de um objeto Pandas de maneiras diferentes.

## Considerações Finais

Renomear as colunas pode ser muito necessário quando se tem um time de pessoas que trabalham em mesmos datasets, facilitando o entendimento interno com o código.
